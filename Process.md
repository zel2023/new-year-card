# 项目开发记录

## 2025-12-31

### 项目启动
- 用户需求：创建一个新年贺卡送给朋友
- 确定方向：互动网页版贺卡
- 确定主题：2026马年（活力橙红、金色配色）

### 功能规划
- 互动网页版设计
- 动画特效（烟花、粒子）
- 音乐背景支持
- 个性祝福语轮播
- 互动功能（抽签、倒计时）

### 开发实施

#### 1. 项目结构搭建
- 创建项目文件夹 `new-year-card-2026`
- 创建必要的子目录：`assets/`, `log/`
- 初始化 TODO.md 计划文档

#### 2. HTML 结构实现
- 封面页设计（开场动画）
- 主贺卡内容区域
- 祝福语展示区
- 互动功能卡片（抽签、烟花、音乐）
- 春节倒计时显示
- Canvas 画布（粒子和烟花）

#### 3. CSS 样式设计
- 马年主题配色方案：
  - 主红色：#D32F2F
  - 主橙色：#FF6F00
  - 主金色：#FFB300
- 渐变背景和光影效果
- 响应式布局（PC + 移动端）
- 多种动画效果：
  - fadeIn（淡入）
  - slideUp（上滑）
  - bounce（弹跳）
  - swing（摆动）
  - gallop（奔跑）

#### 4. JavaScript 交互功能
- **粒子系统**：100个飘落的装饰粒子
- **烟花特效**：Canvas 实现物理烟花效果
- **祝福语轮播**：8条马年祝福语自动切换
- **新年抽签**：5种运势随机抽取
- **春节倒计时**：精确到秒的实时倒计时（目标：2026-01-29）
- **音乐控制**：背景音乐播放/暂停
- **点击互动**：屏幕任意位置触发烟花

### 技术亮点
- 纯前端实现，无需服务器
- Canvas 2D API 绘制特效
- CSS3 动画和过渡效果
- 响应式设计适配多端
- 事件驱动的交互逻辑

### 可自定义配置
- 祝福语内容（script.js）
- 主题配色（style.css）
- 背景音乐（assets/）
- 倒计时目标日期（script.js）
