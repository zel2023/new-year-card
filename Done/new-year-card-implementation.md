# 新年贺卡项目计划

## 项目目标
创建一个精美的互动网页新年贺卡，包含动画特效、音乐背景、个性祝福和多种互动功能。

## 实现计划

### 1. 页面设计与布局 [√]
- 响应式设计，支持PC和移动端
- 2026马年主题配色（活力橙红、金色、中国红）
- 优雅的中国风或现代简约风格
- 可选马的剪影或图案装饰元素

### 2. 核心功能实现 [√]
- **开场动画**：打开贺卡的展开动画效果
- **烟花特效**：点击屏幕触发烟花绽放
- **粒子系统**：飘落的雪花/金币/星星等元素
- **祝福语轮播**：多条祝福语自动切换或点击切换
- **背景音乐**：新年主题音乐，可控制播放/暂停

### 3. 互动功能 [√]
- **点击互动**：点击屏幕任意位置触发特效
- **新年抽签**：抽取新年运势或祝福语
- **倒计时**：距离新年的倒计时动画
- **分享功能**：可以分享给更多朋友

### 4. 技术实现 [√]
- 纯HTML + CSS + JavaScript
- 使用Canvas实现粒子和烟花效果
- 响应式设计，移动端优化
- 无需服务器，可直接打开使用

### 5. 个性化配置 [√]
- 可自定义祝福语内容
- 可替换背景音乐
- 可调整主题色
- 添加朋友的名字

### 6. 测试与优化 [√]
- 浏览器兼容性测试
- 移动端适配测试
- 性能优化
- 添加使用说明

### 7. 文档输出 [√]
- 创建README.md使用说明
- 提供示例运行方式
- 记录Process.md

## 技术栈
- HTML5
- CSS3 (动画、渐变、过渡效果)
- JavaScript (ES6+)
- Canvas API (粒子和烟花效果)
- Web Audio API (音乐播放)

## 交付物
- index.html - 主页面
- style.css - 样式文件
- script.js - 交互逻辑
- assets/ - 资源文件夹（音乐、图片等）
- README.md - 使用说明
- Process.md - 开发记录
