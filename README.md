# 2026马年新年贺卡

一个精美的互动网页新年贺卡，专为2026马年设计，包含动画特效、音乐背景、个性祝福和多种互动功能。

## 功能特色

### 🎆 视觉特效
- **烟花动画** - 点击屏幕任意位置触发绚丽烟花绽放
- **粒子系统** - 飘落的金色粒子装饰，营造节日氛围
- **开场动画** - 优雅的封面展开效果
- **马年主题** - 活力橙红、金色配色，中国风设计

### 💝 互动功能
- **祝福语轮播** - 8条精心设计的马年祝福，点击切换
- **新年抽签** - 趣味互动，抽取新年运势
- **倒计时** - 实时显示距离2026春节的时间
- **烟花秀** - 一键触发多重烟花特效

### 🎵 音乐背景
- 支持背景音乐播放（需自行添加音乐文件）
- 可控制播放/暂停

### 📱 响应式设计
- 完美支持PC和移动端
- 自适应不同屏幕尺寸

## 快速开始

### 方式一：直接打开（推荐）
1. 下载整个项目文件夹

## 个性化定制

### 修改祝福语
编辑 `script.js` 文件中的 `blessings` 数组：
```javascript
const blessings = [
    "你的祝福语 1",
    "你的祝福语 2",
    // ... 添加更多
];
```

### 添加背景音乐
1. 准备一个 MP3 格式的音乐文件
2. 将文件命名为 `new-year-music.mp3`
3. 放入 `assets/` 文件夹
4. 点击"播放音乐"按钮即可

### 修改主题配色
编辑 `style.css` 文件中的 CSS 变量：
```css
:root {
    --primary-red: #D32F2F;      /* 主红色 */
    --primary-orange: #FF6F00;   /* 主橙色 */
    --primary-gold: #FFB300;     /* 主金色 */
    /* ... 其他配色 */
}
```

### 调整春节日期
编辑 `script.js` 中的倒计时目标日期：
```javascript
const newYear = new Date('2026-01-29T00:00:00').getTime();
```

## 使用说明

### 基本操作
1. **打开贺卡** - 点击封面的"打开贺卡"按钮
2. **放烟花** - 点击屏幕任意位置触发烟花
3. **切换祝福** - 点击"换一句祝福"查看不同祝福语
4. **抽签** - 点击"点击抽签"获取新年运势
5. **音乐** - 点击"播放音乐"开启/关闭背景音乐

## 注意事项

1. **音乐文件** - 项目默认不包含音乐文件，需自行添加
2. **浏览器权限** - 部分浏览器可能需要用户交互才能播放音乐
3. **性能** - 大量烟花特效可能影响低端设备性能
4. **网络字体** - 使用系统默认中文字体，无需额外下载

## 常见问题

**Q: 音乐无法播放？**
A: 请确保已将音乐文件放入 `assets/new-year-music.mp3`，且浏览器允许自动播放。

**Q: 烟花效果卡顿？**
A: 减少点击频率，或在低性能设备上避免频繁触发烟花。

**Q: 如何修改倒计时目标？**
A: 编辑 `script.js` 中的 `newYear` 变量，设置目标日期。
